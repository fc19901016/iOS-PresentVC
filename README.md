> 上、左、下、右、中 弹出控制器

| 方向 | 枚举 |
| ------ | ------ |
| 中弹 | XPModalDirectionCenter |
| 上弹 | XPModalDirectionTop |
| 右弹 | XPModalDirectionRight |
| 下弹 | XPModalDirectionBottom |
| 左弹 | XPModalDirectionLeft |
