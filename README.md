# iOS-PresentVC
- 上、左、下、右、中 弹出控制器
- 如果你喜欢的话，给右上角的☆点一下，谢谢！！！

| 弹出方向 | 枚举 |
| ------ | ------ |
| 中 | XPModalDirectionCenter[PlDb] |
| 上 | XPModalDirectionTop[PlGh] |
| 右 | XPModalDirectionRight[PlGd] |
| 下 | XPModalDirectionBottom[PlOd] |
| 左 | XPModalDirectionLeft[PlMe] |
